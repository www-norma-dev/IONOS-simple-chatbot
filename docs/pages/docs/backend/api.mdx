# Backend API

Base URL: `http://localhost:8000`

## Headers
- `x-model-id`: required. Example: `meta-llama/Llama-3.3-70B-Instruct`.

## GET /
## GET /
Returns an empty array (no persistent chat history stored on backend).

Response:
```json
[]
```

## POST /
Sends conversation messages to the ReAct agent. Body:
```json
{ 
  "messages": [
    {"type": "human", "content": "What's the latest news about AI?"},
    {"type": "ai", "content": "Previous AI response..."},
    {"type": "human", "content": "Tell me more about that"}
  ]
}
```
Requires header `x-model-id` (e.g., `meta-llama/Llama-3.3-70B-Instruct`).

Response: Returns the last message from the agent (single message object):
```json
{
  "type": "ai", 
  "content": "Here's the latest information about AI developments..."
}
```
