# Prerequisites

Before you start, make sure you have the following:

- **Python** (v3.10 or higher)
- A dependency manager like **pip**
- **Node.js** (v18 or higher) and **npm** (only if using Next.js frontend)
- An **IONOS API Key** for AI Model Hub access (inference models)
- A **Tavily API Key** for web search functionality
- **(Optional)** An **IONOS Studio API Key** for fine-tuned models
